# Validation rules

The deterministic registry contains ITEM-001–004, UOM-001–002, PKG-001–002, BOM-001–004, REV-001, AVL-001, SUP-001–002, RTG-001, and IMP-001–003. Each result carries severity, explanation, affected record, source evidence, current and candidate values, confidence when applicable, bounded action, and approval requirement.

`BOM-003` is always a hard stop. Duplicate candidates are advisory until a human records a disposition; ambiguous candidates become a hard stop rather than an automatic merge.
